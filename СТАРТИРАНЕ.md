# Как да стартирам AIRIS 1.0

## Локално стартиране

1. **Клонирайте репозиторито:**
```bash
git clone https://github.com/Radilovk/airis1.0.git
cd airis1.0
```

2. **Стартирайте приложението:**
```bash
npm run dev
```

3. **Отворете в браузър:**
   - Отворете http://localhost:5173

## Deployment в GitHub Pages

1. **Създайте production build:**
```bash
npm run build
```

2. **Deploy в GitHub Pages:**
```bash
npm run deploy
```

## Deployment в Netlify/Vercel

### Netlify:
1. Свържете GitHub репозиторито с Netlify
2. Build command: `npm run build`
3. Publish directory: `dist`

### Vercel:
1. Импортирайте проекта от GitHub
2. Framework Preset: Vite
3. Build Command: `npm run build`
4. Output Directory: `dist`

## Изисквания

- Node.js 18+ (препоръчителна версия 20)
- npm или yarn

## Конфигурация

Всички AI модели и настройки се управляват от:
- Административен панел (Settings → AI Models)
- Файл: `runtime.config.json`
