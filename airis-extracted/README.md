# AIRIS Project - Archived Spark Export Files

Тази директория съдържа оригиналните helper файлове от GitHub Spark експорта.

## Файлове

- **DEPLOYMENT_README.md** - Оригинална документация за Spark приложението с описание на всички features
- **.gitignore** - Git ignore правила (копирани в root)
- **extract-project.js** - Node.js скрипт за извличане на файлове от JSON експорт
- **extract-project.py** - Python скрипт за извличане на файлове от JSON експорт

## Забележка

Тези файлове са съхранени за референция. Основният проект сега е пълноценно React/Vite приложение в root директорията.

За актуална документация вижте:
- `/README.md` - Основна документация
- `/CONTRIBUTING.md` - Guidelines за принос

## JSON Export

Оригиналният JSON експорт `airis-full-project-export-2025-11-20.json` в root директорията съдържаше само тези helper файлове и документация, но не и самото приложение.

Проектът е реконструиран като модерно React приложение базирано на спецификацията в DEPLOYMENT_README.md.
